## Setup
Este projeto foi criado com o [Create React App](https://github.com/facebook/create-react-app).  

**adcionando componente de forma global**
```
npm install -g create-react-app
```

**criando o projeto**
```
create-react-app app 
```

**rodando projeto criado **  
```
npm start
```
